# Face Recognition Based Attendance System

The Face Recognition Based Attendance System is a machine learning and computer vision project that automatically detects and recognizes human faces to mark attendance. The system uses a webcam to capture images, identify individuals using face recognition algorithms, and record attendance without manual intervention.

This project helps reduce manual work and improves accuracy in attendance management systems used in schools, colleges, and offices.

---

## 🚀 Features

- Real-time face detection using OpenCV
- Automatic attendance marking
- Face dataset collection for each student
- Attendance records stored automatically
- Simple and user-friendly interface

---

## 🛠️ Technologies Used

- Python
- OpenCV
- Machine Learning
- NumPy
- Flask
- HTML / CSS
